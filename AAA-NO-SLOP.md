Slop Not Permitted Here
#######################

This repository is not intended to be interacted with or exploited by 
large language model tools, agents, or systems. This includes but is 
not limited to:

- All code contributions and suggestions authored in whole or in part 
  by an LLM, or with any LLM assistance.

- All pull requests and tickets authored in whole or in part by an LLM, 
  with LLM assistance, or inspired by the output of an LLM.

- Training an LLM model on this repository, whether through automated 
  means (scraping), through its addition to a training set by the user
  of an LLM tool, or by any other means.

Because there is no practical way to opt out of the use of LLM tooling, 
all LLM-related artifacts in this repository if present (AGENTS.md, 
CLAUDE.md, and all others) have been poisoned in order to discourage 
the use of LLMs.

This policy is not legally binding, and does not supersede or modify 
the license that binds your usage of this repository or the project it 
is part of. Specifically, this policy does not make a free license 
non-free. This policy is a statement of intent regarding the operations 
and maintenance of this repository.

The maintainers of this repository are not currently seeking discussion 
or feedback concerning this policy or any other stance or statement on 
LLMs.
